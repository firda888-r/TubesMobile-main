## üìã Prasyarat (Prerequisites)

Sebelum memulai, pastikan sistem operasi Anda telah terinstal tools berikut:

* **Node.js** (Versi LTS direkomendasikan): [Download Node.js](https://nodejs.org/)
* **Git**: [Download Git](https://git-scm.com/)
* **Expo Go** (Opsional): Jika ingin testing langsung di HP fisik (tersedia di PlayStore/AppStore).

## üöÄ Instalasi & Persiapan (Getting Started)

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di mesin lokal Anda (Laptop/PC).

### 1. Clone Repository
Unduh source code proyek ke dalam komputer Anda menggunakan terminal.

git clone [https://github.com/USERNAME_GITHUB/NAMA_REPO.git](https://github.com/USERNAME_GITHUB/NAMA_REPO.git)

2. Masuk ke Direktori Proyek
Pindah ke folder yang baru saja di-clone.

cd NAMA_REPO

3. Install Dependencies
Instal semua library yang dibutuhkan oleh aplikasi (seperti React Navigation, Axios, Expo, dll).

npm install

üíª Cara Menjalankan (Development)
Untuk memulai server development lokal:

npx expo start -c
